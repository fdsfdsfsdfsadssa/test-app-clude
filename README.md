# 🎮 مدير البلايستيشن v2

## الميزات الجديدة
- **جلسة مفتوحة ∞** — العداد يحسب لأعلى، الزبون يدفع لما يخلص
- **حساب بالنص ساعة** — مثلاً 1ساعة 45دقيقة = 1.5 ساعة مُحتسبة = 15ج
- **لوحة مفاتيح مدمجة** — بدون لوحة مفاتيح الجهاز
- **تعديل الطلبات** — إضافة ساعات على الطلبات المحددة
- **سجل الأيام** — كل الأيام المنتهية محفوظة
- **وضع الاختبار** — 10 ثوان = 1 ساعة للتجربة

## بناء الـ APK عبر GitHub Actions
1. ارفع المشروع على GitHub repository جديد
2. روح **Actions** ← **Build PlayStation Manager APK** ← **Run workflow**
3. بعد ~5 دقائق نزّل الـ APK من **Artifacts**

## تشغيل البرنامج
- **أول تشغيل**: ادخل الإعدادات واضبط سعر الساعة
- **للتجربة**: فعّل "وضع الاختبار" في الإعدادات
