# 🎮 مدير البلايستيشن - PlayStation Manager

برنامج أندرويد لإدارة كافيه البلايستيشن - يعمل على أجهزة Android 5.0+ القديمة والجديدة.

## ✅ المميزات
- تسجيل أجهزة جديدة مع تحديد عدد الساعات
- حساب السعر تلقائيًا
- حساب الباقي للزبون فورًا
- عداد تنازلي لكل جهاز (مع تنبيه عند انتهاء الوقت)
- ملخص إجمالي في نهاية اليوم
- يحفظ البيانات محليًا على الجهاز

## 🚀 كيف تبني الـ APK

### الطريقة: GitHub Actions

1. ارفع المشروع على GitHub repository جديد
2. اذهب إلى **Actions** ← اضغط على workflow **Build PlayStation Manager APK**
3. اضغط **Run workflow**
4. بعد انتهاء البناء (~5 دقائق) انزّل الـ APK من **Artifacts**

## ⚙️ الإعدادات
أول ما تفتح البرنامج، ادخل سعر الساعة والعملة.

## 📱 متطلبات التشغيل
- Android 5.0 (API 21) أو أحدث
